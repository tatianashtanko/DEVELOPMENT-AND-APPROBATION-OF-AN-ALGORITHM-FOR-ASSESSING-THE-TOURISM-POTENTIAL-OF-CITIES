# Разработка и апробация алгоритма оценки туристического потенциала городов в условиях различной степени индустриализации
Оценка туристического потенциала на данный момент является стратегическим направлением, так как многие поселения находятся в поиске путей перепрофилирования, а дальнейшее развитие туризма позволит городам обеспечить новые возможности экономического и социального развития, сохранить историю места, предложить туристам уникальный аутентичный опыт. 
Автор ВКР: Штанько Татьяна Викторовна, Университет ИТМО, Институт дизайна и урбанистики
